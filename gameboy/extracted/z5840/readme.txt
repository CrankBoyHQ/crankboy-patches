# ===================== #
# Patching Instructions #
# ===================== #

1. Find a ROM of Pokémon Crystal. 1.0 or 1.1 works. (I can't help with this part)
2. Go to https://www.marcrobledo.com/RomPatcher.js/
3. Upload the ROM file and the corresponding .ups file. If your Crystal ROM is
   1.0, use crystalleaf.ups. If it's 1.1, use crystalleaf_alt.ups. If you don't
   know which version you have, try both patches. The site will give a warning if
   you use the wrong one.
4. Apply the patch.
5. Load the ROM in your preferred emulator. If you don't have one, I suggest BGB
   or Gambatte.
