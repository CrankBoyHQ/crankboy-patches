Japanese features in international Pokémon Yellow
https://github.com/Masaru2/pokeyellow

This hack restores and fixes some features that were removed or altered from the japanese versions of yellow
https://tcrf.net/Pok%C3%A9mon_Yellow#Version_Differences

Features:
-Fix the "presents" gfx bug where it dosen't appear where it's supposed to
-Restored various texts from the japanese counterpart
-Restored the original town map border
-Restored the JP yellow GFX (Different from blue)
-Compatibility with Super Gameboy, Stadium 1&2 and link trading/battle

Misc:
-Added debug mode as a secondary patch
-Fight menu from debug mode partially translated

Issues:
-Flashing lights couldn't be restored due to an issue with the GBC mode, the flashing lights can work on GB/SGB mode but it lags in GBC mode
